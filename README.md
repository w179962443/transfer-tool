# transfer-tool

一个安卓文件分类搬运工具。用户选择手机上的源目录和目标目录后，应用会把源目录里的文件移动到目标目录，并按“扩展名 / 月份”创建分类目录。

## 当前能力

- 通过系统目录选择器选择源目录和目标目录。
- 防止目标目录与源目录相同，或位于源目录内部。
- 分批发现和传输文件，不会在开始时一次性扫描完整目录树。
- 支持参数 `a`：每批最多发现并传输 `a` 个文件。
- 支持自动连续执行，也支持每完成一批后手动点击“继续下一批”。
- 展示总任务概览、每个批次的完成情况，以及失败任务。
- 支持单个失败任务重试，也支持全部失败任务重试。

## 使用方式

1. 选择源目录。
2. 选择目标目录。
3. 设置“每批最大文件数 a”。
4. 打开“自动连续执行后续批次”时，任务会自动一批接一批执行。
5. 关闭自动模式时，每次点击按钮只执行一批。

## 构建
